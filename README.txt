OUSMANE NIKIEMA — PORTFOLIO SITE
=================================

This is a multi-page static website. To put it live on GitHub Pages:

1. Unzip this folder. Copy ALL of its contents (index.html, the other .html
   files, styles.css, and the images/ folder) into your repository — keeping
   the folder structure exactly as-is. index.html must stay at the top level.
2. Add your resume: drop a file named  resume.pdf  next to index.html and the
   "Resume" link in the top nav lights up automatically.
3. On github.com: repo Settings -> Pages -> Source: "main" branch / root -> Save.
   In ~1 minute it's live. Name the repo  yourusername.github.io  for a clean URL.

THINGS TO EDIT (search the files for "EDIT ME" or "yourusername"):
- GitHub link: replace https://github.com/yourusername everywhere with your URL.
- Project links: each project detail page can link out to a repo/demo — currently
  the gallery cards open the detail pages (that's fine to leave as-is).
- Profile photo (optional): the home page uses an "ON" monogram by default. To use
  a real photo, add images/profile.jpg and follow the comment in index.html.
